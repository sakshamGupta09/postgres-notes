# Order By

- To sort the data in our table by one or more columns.
- First `From` runs, then `WHERE`, then `SELECT` and then `ORDER BY`. Hence, we can use aliases also for sorting.
- Default order is `ASC`

```sql
SELECT
  *
FROM Table
ORDER BY
  C1 [ASC | DESC],
  C2 [ASC | DESC]
```

```sql
SELECT
	 first_name,
 	last_name,
	 gender,
	 date_of_birth AS "dob"
FROM actors
ORDER BY
	 gender ASC,
	 dob DESC;
```

```sql
SELECT
	first_name,
	last_name,
	LENGTH(first_name) AS "length"
FROM actors
ORDER BY
	length DESC;
```

## Order by column number

- Order by runs after `select` and `where`. So, It always runs on a result set.
- Now instead of sorting by column names, we can instead sort by column order in the resulting set.

```sql
SELECT
	first_name,
	last_name,
	LENGTH(first_name) AS "length"
FROM actors
ORDER BY
	3 DESC;
```
