# Aliasing columns

- As a techie, we might be following a specific nomenclature to name our table columns.
- But for other people that nomenclature might not make too much sense. So, We should be able to customize column names while reading data.
- In some cases, we want to perform some operations like get full name from first and lastname. In that case our column gets assigned a weird name automatically.


```sql
SELECT
  C1 AS "SOME_OTHER_NAME"
FROM Table
```
- It's better to use double quotes. Single quotes doesn't work and gives an error.


## As keyword is optional

- But still for readability and intent it's better ti use it.

```sql
SELECT
  C1 "SOME_OTHER_NAME"
FROM Table
```
