# Distinct

- I want only the unique values of this column in the resulting set.

```sql
SELECT
	DISTINCT movie_lang AS "movie_lang"
FROM movies
ORDER BY
	movie_lang ASC;
```

## Multiple distinct column values

- I want only the unique combination of firstName and lastName.

```sql
SELECT
	DISTINCT first_name, last_name
FROM actors
ORDER BY
	first_name ASC;

```
