# Where Clause

- I do not want all the rows in my table. I want only those rows whose coluumns satisfies a condition.
- The conditions are written using operators. We have 3 categories of operators.

## Comparison

- Compare the column value with another value using `=`, `<>`, `>=`, `<=`.


## Logical

- `AND`, `OR`, `NOR`, `NOT`, `LIKE`, `IN`, `BETWEEN`.

## Arithmetic

- `+`, `-`, `*`, `/`, `%`.


## Comparison Example

- Here we are filtering by comparison. That value should be equal to English.
- Values must be mentioned in single quotes.

```sql
SELECT
	*
FROM movies
WHERE
	movie_lang = 'English';
```

## Multiple conditions with AND

- Give me all the English movies with age certificate equals 18.
- Here we are combining two comparison queries using `AND`.
- Both the expressions hsould return true for a row to be selected.

```sql
SELECT
	*
FROM movies
WHERE
	movie_lang = 'English' AND
	age_certificate = '18';
```


## OR

- Either one of the expression should be true and that row will be part of result set.

```sql
SELECT
	*
FROM movies
WHERE
	movie_lang = 'English' OR
	movie_lang = 'Chinese'
ORDER BY 
	movie_lang ASC;
```

## Combining AND with OR

- Give all chinese or english movies with age certificate of 12.

```sql
SELECT
	*
FROM movies
WHERE
	movie_lang = 'English' OR
	movie_lang = 'Chinese' AND
	age_certificate = '12'
ORDER BY 
	movie_lang ASC;
```
- AND operator takes higher precedence.
- This goes to every row and asks if any there is ay movie for which either the language is english or language is chinese and age_certificate is 12. It return either english movies or chinese with age certificate of 12.
- So, we need to use parenthesis.

```sql
SELECT
	*
FROM movies
WHERE
	(movie_lang = 'English' OR movie_lang = 'Chinese') AND
	age_certificate = '12'
ORDER BY 
	movie_lang ASC;
```
