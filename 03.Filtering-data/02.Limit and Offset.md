# Limit

- To limit number of records that we get from the DBMS server.
- Less data is bieng transferred over the wire and less resources are consumed.
- `LIMIT` clause is written at the end.
- Useful in top n type of queries.

```sql
SELECT
	*
FROM movies
WHERE
	movie_lang = 'English'
ORDER BY
	movie_length DESC
LIMIT 5
```

```sql
SELECT
	*
FROM movies
ORDER BY
	movie_length DESC
LIMIT 5;
```

## Offset

- To implement pagination.
- Let's say as a result of a query, we get 1 million resulting rows.
- Now we will keep sending 20 records at a time and client will ask for next set of 20 records.
