# Insert data in table

- I want to insert one row in my table and provide values for these columns.
- Values should always be in a single quote for strings.

```sql
INSERT INTO table_name (first_name, last_name, email, age)
VALUES ('John', 'Doe', 'john@example.com', 28);
```


## Insert multiple rows

- In this table, for these columns, add multiple rows with these values.
- After `VALUES`, we supply a row.

```sql
INSERT INTO tableName (C1, C2, C3)
VALUES
(V1, V2, V3),
(V1, V2, V3),
(V1, V2, V3),
```

## Returning

- By default after running an insert statement, the DB gives us a generic messsage that operation was successful.
- Sometimes, we instead want the row that we inserted in response to query.
- `*` will return all the columns added. We can instead add custom ccolumns.

```sql
INSERT INTO table (C1, C2, C3)
VALUES
(V1, V2, V3)
RETURNING *
```
