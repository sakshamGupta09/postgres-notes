# Update

- First, we need to filter rows which needs to be updated using a `where` clause.
- Go to these rows, and update these column values.

``` sql
UPDATE tableName
SET
  C1=V1,
  C2=V2,
  C3=V3
WHERE
  condition
```

## Returning

- After we execute a query to update a row, we get a generic message "Query returned successfully".
- Instead I want the updated row back in the response.

```sql
``` sql
UPDATE tableName
SET
  C1=V1,
  C2=V2,
  C3=V3
WHERE
  condition
RETURNING *
```
